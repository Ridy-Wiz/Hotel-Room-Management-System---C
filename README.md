# Hotel Management System (C)

## 📌 Description
This is a console-based hotel room booking system written in C.

## 🚀 Features
- Add and manage rooms
- Book rooms with date & time
- Check-out system
- File-based storage (rooms.txt, booked.txt)

## ▶️ How to Run
```bash
gcc main.c -o hotel
./hotel
```

## 📂 Files
- main.c → source code
- rooms.txt → room data
- booked.txt → booking data
